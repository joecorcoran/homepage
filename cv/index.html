Pannier

Source on GitHub • Documentation on Relish

A general-purpose Ruby asset processing tool, it can be used both at the command line and from within any Rack application. Intended as a replacement for the asset pipeline, it is completely decoupled from Rails and provides a Rack-like interface for attaching tools like preprocessors and compressors.

Judge

Source on GitHub

Judge addresses the problem of duplicate form validation code by porting the inbuilt Rails form validation methods to JavaScript and exposing your model validations as JSON within HTML5 data attributes. It also provides an interface for adding custom validators to both client and server where the standard validators aren't enough.
